1. Resumen Ejecutivo

Chuchurex Astral es una aplicación web completa para el cálculo e interpretación de cartas natales astrológicas con ciclos de biorritmos basados en las enseñanzas de Ra (Material de la Ley del Uno).

Características Principales

  • ✓ Cálculos astronómicos de precisión con Swiss Ephemeris
  • ✓ 10 planetas y 12 casas astrológicas
  • ✓ 290+ interpretaciones personalizadas en español
  • ✓ 4 ciclos de biorritmos (incluido Ciclo Espiritual de 18 días)
  • ✓ Soporte multilenguaje (inglés, español, portugués)
  • ✓ Accesibilidad WCAG AAA (contraste 7:1)
  • ✓ Gráfico SVG interactivo de la carta natal
  • ✓ Exportación a PDF

URLs de Producción

  • Frontend: https://chuchurex.cl
  • API Backend: https://api.chuchurex.cl
  • Repositorio: https://github.com/chuchurex/astro-chart

  • CORS configurado para dominios específicos
  • Middleware stack modular

Swiss Ephemeris

  • Motor de cálculo astronómico de alta precisión
  • Usado por astrólogos profesionales
  • Precisión de arco-segundo
  • Integrado vía biblioteca Kerykeion

3. Geocodificación Automática

async function geocodeCity(cityName) {
    const response = await fetch(
        `${CONFIG.NOMINATIM_URL}?city=${encodeURIComponent(cityName)}&format=json`
    );
    const data = await response.json();
    if (data.length > 0) {
        return {
            latitude: parseFloat(data[0].lat),
            longitude: parseFloat(data[0].lon)
        };
    }
    // Fallback: Santiago, Chile
    return { latitude: -33.4489, longitude: -70.6693 };
}

10. Biorritmos y Enseñanzas de Ra

Ciclos Estándar (Basados en Seno)

1. Ciclo Físico (23 días)

  • Energía vital y resistencia física
  • Capacidad de recuperación
  • Rendimiento deportivo

2. Ciclo Emocional (28 días)

  • Estado anímico y sensibilidad
  • Creatividad y expresión artística
  • Relaciones interpersonales

3. Ciclo Intelectual (33 días)

  • Claridad mental y concentración
  • Capacidad de análisis y lógica
  • Memoria y aprendizaje

Ciclo Espiritual/del Adepto (18 días)

Basado en las enseñanzas de Ra del Material de la Ley del Uno (L/L Research).

Análisis de Días Críticos

El sistema analiza automáticamente si múltiples ciclos están en puntos críticos simultáneamente:

  • NIVEL BAJO: 0-1 ciclos en punto crítico
  • NIVEL MODERADO: 2 ciclos en punto crítico
  • NIVEL ALTO: 3-4 ciclos en punto crítico (requiere especial atención)

Visualización en Frontend

Los biorritmos se muestran con:

  • Barras de progreso coloreadas (verde/rojo según positivo/negativo)
  • Porcentaje numérico (-100% a +100%)
  • Indicadores de tendencia (📈 ascenso / 📉 descenso)
  • Descripción de la fase actual
  • Cita de Ra correspondiente (para ciclo espiritual)

13. Desarrollo Local

Requisitos

  • Python 3.14 (o 3.12+)
  • SASS (para compilar estilos)
  • Git
  • Navegador moderno (Chrome, Firefox, Safari, Edge)

Setup Inicial

# 1. Clonar repositorio
git clone https://github.com/chuchurex/astro-chart.git
cd astro-chart

# 2. Crear y activar entorno virtual
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# o en Windows: venv\Scripts\activate

# 3. Instalar dependencias Python
pip install -r requirements.txt

# 4. Crear archivo .env (para deploy)
echo "FTP_PASSWORD=tu_password_aqui" > .env

Ejecutar Backend

# En terminal 1
source venv/bin/activate
uvicorn app:app --reload --port 8001

# El servidor estará en http://localhost:8001
# Swagger docs en http://localhost:8001/docs

Ejecutar Frontend

# En terminal 2
python -m http.server 3000

# El frontend estará en http://localhost:3000

Compilar SASS

# Compilación única
sass styles.scss styles.css

# Watch mode (recompila automáticamente)
sass styles.scss styles.css --watch

Estructura de Desarrollo

Terminal 1: Backend API (puerto 8001)
Terminal 2: Frontend Server (puerto 3000)
Terminal 3: SASS Watcher (opcional)
Terminal 4: Git, comandos generales

Testing

# Test de carta de ejemplo
curl http://localhost:8001/chart/example

# Test de health check
curl http://localhost:8001/health

# Test manual con navegador
# Abrir http://localhost:3000
# Completar formulario y calcular carta

Hot Reload

  • Backend: Uvicorn con --reload detecta cambios en app.py
  • Frontend: Refrescar navegador manualmente (F5)
  • CSS: SASS watch recompila automáticamente, luego refrescar navegador
